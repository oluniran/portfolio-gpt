# Project rules for this portfolio site

See `CLAUDE.md` for current site structure and the design patterns built up this session
(checkerboard metric tiles, logo-strip marquee, testimonial grid, etc.) — read it before
making changes so new work matches existing conventions instead of drifting.

Standing content-accuracy rule (applies to any future edits, not just this session's):
- Never present a stock/generic photo as if it depicts a specific real client's facility.
- Never fabricate customer quotes, reviewer names, companies, or star ratings. If a real
  quote or stat is needed, source it from the actual company's own public materials
  (their live website, published case studies) and quote it verbatim.
- Case-study "brand voice" copy (callouts, headings, positioning language) should be
  sourced from the person's real internal strategy documents or the real company site —
  never invented to sound plausible.
